# React ToDoList App

This is a simple ToDoList application built using **React**.  

The project was created to practice **React fundamentals** and **web development best practices**.  

---

##  Features

- Add, delete, and mark tasks as completed
- Responsive design
- Real-time updates using React state
- Organized project structure

---

##  What I Learned

During the development of this project, I improved my skills in:

- **React Components** – Functional and reusable components
- **State Management** – Using `useState` to manage dynamic data
- **Event Handling** – Handling user inputs and events
- **Props** – Passing data between components
- **Conditional Rendering** – Rendering elements based on conditions
- **Git & GitHub** – Version control and pushing projects online
- **.gitignore** – Ignoring unnecessary files (node_modules, build, .env)
- **Project Structure** – Organizing React project folders and files
- **Debugging** – Inspecting and fixing errors in the console
